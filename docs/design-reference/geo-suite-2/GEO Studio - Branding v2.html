04

Las dos paletas que te gustan

app-ready · con vista en producto
A · Grafito + acento
Base neutra grafito muy sobria (modo claro), con un único color de acento. El acabado más “Stripe/Linear”. Elige el acento:
B · Negro + un color
El extremo audaz: superficies casi negras (modo oscuro) y un acento vívido que “canta”. Máxima personalidad y diferenciación frente a la competencia.

Recomendación: Grafito + índigo como base diaria (claro, fiable) y Negro + cian/lima como modo oscuro y para la marca/landing (impacto). Es un sistema dual coherente.
